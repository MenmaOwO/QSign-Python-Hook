API 文档

接口说明

路径： /sign

方法： POST

支持客户端：

  • ✓ com.tencent.mobileqq:MSF

请求参数

参数名 类型 必填 说明
cmd string 必需 命令类型（如 wtlogin.login）
seq string 必需 请求序列号（数字字符串）
buffer string 必需 十六进制数据
uin string 必需 用户唯一标识
qua string 必需 客户端标识（如 V1_AND_SQ_8.9.53_3362_YYB_D）
package string 必需 目标包名（必须为 com.tencent.mobileqq:MSF）

请求示例

curl -X POST http://localhost:5000/sign \\
-H "Content-Type: application/json" \\
-d '{
    "cmd": "wtlogin.login",
    "seq": "123456",
    "buffer": "00010203040506070809",
    "uin": "10001",
    "qua": "V1_AND_SQ_8.9.53_3362_YYB_D",
    "package": "com.tencent.mobileqq:MSF"
}'

重要说明

  • ▶ 所有参数值必须为字符串类型
  • ▶ buffer 和 seq 必须为合法十六进制字符串
  • ▶ 建议设置请求超时时间为 5-10 秒
  • ▶ 遇到 400 错误请检查参数格式
  • ▶ 服务端时间与客户端时间差不应超过 5 分钟
